# ✉️ AI Gmail Reply Extension

An AI-powered Chrome Extension that integrates directly into Gmail and generates smart, context-aware email replies with selectable tone using a Java Spring Boot backend and LLM APIs.

## Features
- Generate AI-powered email replies directly inside Gmail
- Select reply tone (Professional, Friendly, Short, Apologetic)
- Regenerate and refine AI responses
- Insert replies as editable drafts (no auto-send)
- Works with Gmail’s dynamic UI

## Tech Stack
- JavaScript (Chrome Extension)
- Java + Spring Boot
- REST APIs
- LLM APIs (Gemini / OpenAI)

## Setup

### Backend
mvn spring-boot:run

### Chrome Extension
1. Open chrome://extensions
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select the extension folder
